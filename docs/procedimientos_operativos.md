
# Procedimientos Operativos (Requests & Pases)

1. **Solicitud de Alta/Change** (plantilla):
   - Descripción del proceso
   - Owner/Soporte
   - Entradas/Salidas (datasets)
   - Calendario/SLAs
   - Dependencias (upstream/downstream)
   - *Checklist* de seguridad y validación

2. **Checklist pre‑prod**:
   - Variables y secretos definidos
   - GE suite passing
   - Data model actualizado
   - DAG habilitado y documentación

3. **Plan de reversión** y *runbook* de incidentes.
